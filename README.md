## File Description

### 1. data_structures_assignment.py
Main file used to run all tasks.  
This file imports and executes Task 1 to Task 4 in sequence.

### 2. task1_products.py
Task 1: Product Collections  
- Uses lists and tuples  
- Creates product list  
- Creates sample product tuple  
- Performs basic list and tuple operations  

### 3. task2_categories.py
Task 2: Categories  
- Uses sets  
- Creates a category set  
- Demonstrates adding elements  
- Checks for existence  
- Counts unique categories  

### 4. task3_pricing.py
Task 3: Product Pricing  
- Uses dictionaries  
- Adds, updates, and removes products  
- Calculates average price  
- Finds minimum and maximum priced products  

### 5. task4_combined.py
Task 4: Combined Operations  
- Combines list, tuple, and dictionary  
- Creates product catalog  
- Groups products by category  
- Finds category with maximum products
